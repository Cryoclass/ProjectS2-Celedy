Voici le Read Me. (on vous le met car on a retenue que c vous tenez a coeur en TP)
Le rapport de soutenance est en soit un immense README donc celui la semble un peu désuhet.

En revanche, nous allons indiquer ici les commandes :

echap : met le jeu sur pause
fleches directionnelles : dirigent le personnages
ZQSD : touches pour donner la direction du tir
A : interagir (se coller avec le pnj rouge puis A pour lui parler)
click gauche : permet de cliquer sur les case dialogues et les menus et l'inventaire.

ATTENTION:
- Aller dans le menu puis revenir ''reset'' la scene, donc le jeu. 
- On ne peux parler qu'une seule fois au même PNJ

Je pense que vous avez toutes les indications pour jouer, bonne découverte, certe un peu courte, mais agréable je l'espère.
